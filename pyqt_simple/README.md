## Quick start

Windows (from the `pyqt_simple` folder):

```bat
python -m pip install -r requirements.txt
python main.py
```

Linux/macOS:

```bash
python3 -m pip install -r requirements.txt
python3 main.py
```

The app opens a small window with a label, a button (increments a counter) and a text field (press Enter to show a message box).
